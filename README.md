# bootstrap

### Installation

Gemfile

```ruby
gem 'bootstrap', git: 'https://github.com/IngateFuture/bootstrap.git'
gem 'compass-rails'
gem 'sass-rails'
```

## Использовать миксины как миксины компаса

```
@import bootstrap

.button
  +button
```

## Миксины тут

```
bootstrap/lib/stylesheets/mixins/
```
